- 根模块作为应用入口，在 `main.dart` 中初始化全局状态容器（ProviderScope）并注入共享偏好存储，协调主题、认证及路由策略。
- 通过条件导入（`if (dart.library.html)`）与平台检测（`kIsWeb`）抽象底层差异，确保 SQLite (Drift) 与媒体服务在 Android 和 Web 端的无缝切换。
- 集成内部包 `reliable_websocket` 处理高可靠性实时通信，该包独立维护连接管理、消息发件箱及同步恢复逻辑，与应用层业务解耦。
- 采用分层架构，由 `lib/services` 提供统一数据访问接口，屏蔽底层 API 客户端与本地数据库的实现细节，供 `lib/providers` 中的状态管理器调用。
